Surround Seven

Hex Fill-a-Pix — each number = green hexes in its 7-cell cluster (itself + its 6 neighbours)
Left-click / drag = paint green  •  Right-click / drag = mark empty (red)  •  click again to clear
Each clue is a number colour-coded to its planetary metal (see key above). Hit Show Symbols for the pure alchemical glyphs.
Symbols fade once every cell in their cluster is decided (greens placed + rest red).
Hexes have 6 neighbours, so clues range 0–7. Every puzzle is solvable by pure logic — no guessing.
Marking empties is essential — a satisfied clue forces its remaining cells empty, which unlocks the next greens. Stuck? Next Step always finds a forced move.

How the hex puzzle is generated
  • Board: a hexagon of hex cells in axial coordinates. Each cell's "cluster" is itself plus its 6 axial neighbours (fewer on the rim).
  • Random solution → clues: the grid is filled randomly; each cell's clue is the count of green cells in its 7-cell cluster.
  • Greedy removal under a logic solver: clues are removed one by one and kept removed only if a deduction-only solver can still rebuild the whole board (rules: cluster's remaining need == its unknowns → all green; cluster already satisfied → rest empty).
  • Result: a unique, no-guess puzzle. This is the exact same engine as the square version — only the adjacency graph changed from 8 neighbours to 6.
