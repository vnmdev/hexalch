Surround Nine

A Fill-a-Pix clone — each number = green cells in its 3×3 block (itself + 8 neighbours)
Left-click / drag = paint green  •  Right-click / drag = mark as empty (dot)  •  click again to clear
Every puzzle is generated to be solvable by pure logic — no guessing required.
How puzzle generation works (the logic)

1. Random solution. A grid is filled randomly (each cell green/empty by the chosen density). This is the secret answer.

2. Compute every clue. For each cell, count green cells in its 3×3 block (including itself). That number is its clue. A grid showing all clues is always solvable, but cluttered.

3. Greedy clue removal under a logic solver. Clues are visited in random order and each is tentatively removed. After removal a deduction-only solver re-solves from scratch using just two rules:

  • If a clue's remaining count equals its unknown cells → all unknowns are green.
  • If a clue is already satisfied by known-green cells → all unknowns are empty.

If the solver still reconstructs the full grid, the clue is genuinely redundant and stays removed; otherwise it's restored. Because complete forced deduction admits exactly one answer, the result is guaranteed unique and human-solvable without guessing.

The same solver powers the Hint button (it applies one forced move) and validates your win.
